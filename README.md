# stacks-lab

Stacks Lab is an experimental workspace for building, testing, and prototyping
applications on the Stacks blockchain.

This repository serves as a flexible environment for exploring smart contracts,
tooling, integrations, and ideas within the Stacks ecosystem.

## Scope
- Prototyping Stacks-based applications
- Exploring ecosystem tools and SDKs
- Testing new ideas and architectural patterns

## Tech Stack
- Clarity
- Clarinet
- Stacks.js
- Bitcoin / Stacks tooling

## Status
Active development and experimentation.
